# 🟡 Pac-Man

A faithful recreation of the classic Pac-Man arcade game built with HTML5 Canvas and vanilla JavaScript.

## Play

👉 [Play on GitHub Pages](https://mjwoolley.github.io/pacman/)

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000/pacman/

## Controls

- **Arrow keys** — Move Pac-Man

## Tech

- HTML5 Canvas
- Vanilla JavaScript (ES modules)
- Vite (dev server + build)
